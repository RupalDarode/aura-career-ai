## ⚙️ Setup
```bash
git clone https://github.com/RupalDarode/aura-career-ai.git
cd aura-career-ai
pip install -r requirements.txt
streamlit run app.py
```

Add to `.streamlit/secrets.toml`:
```toml
GROQ_API_KEY = "your_key_here"
```
